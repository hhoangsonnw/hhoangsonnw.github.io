# Product Context

## Surface

Personal cybersecurity portfolio and write-up archive.

## Audience

Security peers, recruiters, CTF players, and incident-response practitioners who want to inspect practical evidence trails quickly.

## Register

Brand surface / personal portfolio. The design is part of the signal: it should feel like a private evidence vault rather than a generic SaaS dashboard.

## Core Jobs

- Browse write-ups by platform and branch.
- Search and filter published notes.
- Read long Markdown investigations with images, tables, code, and a table of contents.
- Preserve a maintainable static publishing workflow.

## Voice

Private, exacting, forensic, cinematic. The interface should feel controlled and deliberate, with restraint in motion and strong evidence readability.
